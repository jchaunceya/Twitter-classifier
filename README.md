# Twitter-classifier

This is a self contained virtual environment to run the twitter classifier.
On MacOS, this is activated by executing the command 

   'source bin/activate' 

in the sent_env directory.

Python 3.5 packages installed:

       nltk
       tweepy	
       mysqlclient
       numpy
       scipy
       scikit-learn
